## 📋 Описание

Ninebot ES Controller — это система управления электросамокатом Ninebot ES с помощью протокола UART, построенная на микроконтроллере ESP8266. Система предоставляет полный контроль над самокатом через удобный веб-интерфейс, доступный по Wi-Fi.

> ⚠️ **Проект находится в разработке!**  
> Данная версия является **альфа-версией** и предназначена для тестирования и разработки. Возможны баги, несовместимости и изменения в функциональности. Используйте на свой страх и риск!

### ✨ Основные возможности

🔓 **Управление самокатом:**
- Блокировка/разблокировка
- Управление двигателем
- Переключение режимов (Normal/Eco/Sport)
- Ограничение скорости (15-35 км/ч)
- Круиз-контроль

💡 **Освещение и звук:**
- Управление фарами
- Звуковые сигналы
- 10 режимов LED-подсветки
- Настройка цветов (4 канала)

📊 **Мониторинг:**
- Состояние батарей
- Температуры (корпус, батареи, MOS)
- Электрические параметры
- Статистика поездок
- Диагностика ошибок

🌐 **Веб-интерфейс:**
- Адаптивный дизайн
- Мобильная совместимость
- Реальное время обновления
- 6 тематических вкладок

## 🔌 Схема подключения

```
Ninebot ES Board                    ESP8266 (Huzzah)
┌─────────────────┐                ┌─────────────────┐
│                 │                │                 │
│  GND ───────────┼────────────────┼─ GND            │
│                 │                │                 │
│  RX (или TX) ───┼────────────────┼─ GPIO12/D6      │
│                 │                │                 │
│  TX (или RX) ───┼────────────────┼─ GPIO13/D7      │
│                 │                │                 │
│  3.3V ──────────┼────────────────┼─ 3.3V           │
│                 │                │                 │
│  Кнопка ────────┼────────────────┼─ GPIO5/D1       │
│                 │                │                 │
└─────────────────┘                └─────────────────┘
```

⚠️ **ВАЖНО**: Определите правильную полярность RX/TX. Обычно:
- **TX самоката** → **GPIO13 (D7) ESP8266**
- **RX самоката** → **GPIO12 (D6) ESP8266**

### Подключение кнопки (опционально)
Для ручного переключения блокировки:
```
Кнопка ──┬── GPIO5 (D1)
         │
       GND
```

## ⚙️ Настройка

### Первоначальная настройка
1. **Подключение**: Установите ESP8266 согласно схеме
2. **Питание**: Самокат должен быть включен
3. **Wi-Fi**: Подключитесь к сети `NinebotESx` с паролем `12345678`
4. **Доступ**: Откройте браузер и перейдите на `192.168.4.1`

### Кастомизация настроек
В файле `src/main.cpp` можно изменить:

```cpp
// Настройки Wi-Fi (строки 12-13)
const char* ssid = "NinebotESx";          // Имя Wi-Fi сети
const char* password = "12345678";        // Пароль Wi-Fi

// Пин для кнопки (строка 166)
const int BUTTON_PIN = 5;                 // GPIO5/D1

// Интервал heartbeat (строка 169)
const unsigned long HEARTBEAT_INTERVAL = 4000; // 4 секунды
```

### Настройка Serial портов
Если самокат не отвечает, возможно нужно изменить пины в `src/main.cpp`:

```cpp
// Добавить в setup() для изменения Serial
Serial.end();          // Закрыть текущий Serial
Serial.begin(115200, SERIAL_8N1, 13, 12); // RX=GPIO13, TX=GPIO12
// или
Serial.begin(115200, SERIAL_8N1, 12, 13); // RX=GPIO12, TX=GPIO13
```

#### 🔒 Блокировка/разблокировка

- **Заблокировать**: Кнопка "Заблокировать" или `/lock`
- **Переключить**: Кнопка "Лимит скорости" или `/toggle`

#### 🎛️ Режимы работы
- **NORMAL**: Стандартная мощность
- **ECO**: Экономичный режим
- **SPORT**: Максимальная мощность

#### 📏 Ограничение скорости
- **15 км/ч**: Безопасный режим
- **20 км/ч**: Городской режим
- **25 км/ч**: Стандартный лимит
- **30-35 км/ч**: Максимальная скорость

#### 💡 Подсветка
- **10 режимов**: От простого свечения до полицейских мигалок
- **4 цвета**: Независимая настройка каждого канала
- **Эффекты**: Дыхание, мерцание, последовательное переключение

### API эндпоинты

#### Основное управление
- `GET /unlock` - Разблокировать
- `GET /lock` - Заблокировать
- `GET /toggle` - Переключить блокировку
- `GET /status` - Получить статус

#### Режимы работы
- `GET /mode_normal` - Режим Normal
- `GET /mode_eco` - Режим Eco
- `GET /mode_sport` - Режим Sport

#### Скорость
- `GET /speed_15` - Лимит 15 км/ч
- `GET /speed_20` - Лимит 20 км/ч
- `GET /speed_25` - Лимит 25 км/ч
- `GET /speed_30` - Лимит 30 км/ч

#### Функции
- `GET /headlight_toggle` - Переключить фары
- `GET /beep_toggle` - Переключить звук
- `GET /cruise_toggle` - Переключить круиз-контроль

#### Данные
- `GET /data` - Все данные самоката
- `GET /extended` - Расширенная информация

### Протокол связи

Система использует оригинальный протокол Ninebot ES:

#### Структура пакета
```
[Header] [Length] [Type] [Index] [Data...] [Checksum]
  5A A5     02        01     22      00 00     F0 F0
```

#### Типы команд
- **0x01**: Чтение регистра (CMD_CMAP_RD)
- **0x02**: Запись регистра (CMD_CMAP_WR)
- **0x03**: Запись без ответа (CMD_CMAP_WR_NR)
- **0x55**: Heartbeat

#### Основные регистры
| Адрес | Название | Описание | Размер |
|-------|----------|----------|---------|
| 0x22 | INDEX_BATTERY | Емкость батареи (%) | 16 бит |
| 0x26 | INDEX_SPEED | Скорость (0.1 км/ч) | 16 бит |
| 0x70 | INDEX_LOCK | Блокировка | 16 бит |
| 0x71 | INDEX_UNLOCK | Разблокировка | 16 бит |
| 0x75 | INDEX_WORK_MODE | Режим работы | 16 бит |
| 0x7C | INDEX_CRUISE | Круиз-контроль | 16 бит |

### Системные требования

#### ESP8266
- **Flash память**: 4MB (минимум 2MB для LittleFS)
- **Оперативная память**: 80KB (используется ~40KB)
- **GPIO**: 3 пина (опционально 4)
- **Питание**: 3.3V, 250mA максимум

#### Производительность
- **Время отклика**: <100ms
- **Обновление данных**: 3 секунды
- **Heartbeat**: каждые 4 секунды
- **Wi-Fi клиенты**: до 10 устройств

### FAQ

**Q: Можно ли использовать без кнопки?**  
A: Да, кнопка опциональна. Все управление доступно через веб-интерфейс.

**Q: Работает ли с другими моделями Ninebot?**  
A: Пока только с Ninebot ES. Другие модели требуют адаптации протокола.

**Q: Можно ли питать ESP8266 от USB?**  
A: Да, но для работы с самокатом требуется питание 3.3V от самоката.

**Q: Безопасно ли это для самоката?**  
A: Система использует только официальные команды протокола Ninebot.
